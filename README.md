## 使用方式

查看实时log网页:

- 运行Nonebot实例
- 浏览器访问 http://127.0.0.1:你的Nonebot端口/logs 

查看log输出流(SSE):

- 运行Nonebot实例
- 浏览器访问 http://127.0.0.1:你的Nonebot端口/logs/sse
